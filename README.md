# Gangsta eggs

![](https://gangsta-eggs.com/1.gif)

Gangsta Eggs is a NFT collection deployed on the Polygon sidechain, aimimg at providing a gamified experience of mintable NFTs.

# Quickstart

## Installation

- `npm install`
- `cd frontend && npm install`
